CSV file opener and reader

03/2020

Two of my courseworks for Object Oriented programming module were to build a .csv file opener and reader. The difference between these courseworks was that one was a desktop app and the other was a browser application.

I started off with the desktop application. The first thing I needed to do was to build a data structure that would be able to hold the data being read in from the .csv file. The data structure I decided to use was a DataFrame that was made up of Columns objects. Each column object represented a field in the .csv file.
